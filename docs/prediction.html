2 Feature Selection

Can tipping behavior in NYC yellow taxi credit card transactions be predicted based on trip characteristics using linear regression?

During exploratory analysis, scatterplots and boxplots were reviewed for linear relationships against tip_amount. Further investigation revealed total_amount to be the best statistical predictor. Total_amount is a function of all other variables, but mostly trip_distance and trip_duration_min. While we could use total_amount as the single best predictor it does not provide us with practical prediction for customers, executives or investors. Customers know the average tip amount is 17% and will likely tip between 10-20%. Executives want to ensure their employees are supporting in a thriving business of taxi service. Investors need insights. These models will focus on a predictive model for executives which could be applied for business analytics such as compensation reviews or contractor negotiations.

# Calculate VIF where feasible
get_vif_summary <- function(model, name) {
  if (name == "Full Model") {
    return(tibble(
      Model = name,
      `Max VIF` = NA_real_,
      `Mean VIF` = NA_real_,
      `VIF > 5` = NA_integer_,
      `VIF > 10` = NA_integer_,
      `Total Predictors` = length(coef(model)) - 1,
      Notes = "Too complex (includes PULocationID + DOLocationID)"
    ))
  }
  
  vifs <- tryCatch({
    vif_values <- vif(model)
    if (is.matrix(vif_values)) {
      # For models with categorical variables (GVIF)
      vif_values[, "GVIF^(1/(2*Df))"]
    } else {
      vif_values
    }
  }, error = function(e) rep(1, length(coef(model)) - 1))
  
  tibble(
    Model = name,
    `Max VIF` = round(max(vifs, na.rm = TRUE), 2),
    `Mean VIF` = round(mean(vifs, na.rm = TRUE), 2),
    `VIF > 5` = sum(vifs > 5, na.rm = TRUE),
    `VIF > 10` = sum(vifs > 10, na.rm = TRUE),
    `Total Predictors` = length(vifs),
    Notes = ""
  )
}

# Helper functions for test set evaluation
rmse <- function(actual, predicted) {
  sqrt(mean((actual - predicted)^2, na.rm = TRUE))
}

mae <- function(actual, predicted) {
  mean(abs(actual - predicted), na.rm = TRUE)
}

rsq <- function(actual, predicted) {
  1 - sum((actual - predicted)^2, na.rm = TRUE) / 
      sum((actual - mean(actual, na.rm = TRUE))^2, na.rm = TRUE)
}
